My Projects.

Obstacle Avoiding Robot

The purpose of this project was to learn Arduino and implement the programming concepts to control different sensors and hardwares. This robot was made basically to avoid any obstacle in front of it. If any obstacle is detected by the ultrasonic sensor (attached on top of Servo motor) in its path it sends the signal to the microcontroller (Arduino Uno board). Which calculates the distance and controls the Servo motor to turn in a right and left direction to see the available path and moves accordingly.
